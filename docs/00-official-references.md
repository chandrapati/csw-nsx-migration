## Cisco companions for the deltas
- Cisco Secure Firewall (IDS/IPS, L7/app control) — pair for NSX Distributed IDS/IPS + Context Profiles
- Cisco Hypershield — modern L7/segmentation for the deltas
- Cisco ISE — user/device identity to substitute NSX Identity Firewall

> Behaviors, object names, and capabilities change by release for both products. Always confirm against the guides for the exact NSX and CSW versions in the engagement.
